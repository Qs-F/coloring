# package coloring

This is originally private package.  
Maintainer: @CreatorQsF

## Installation

`go get github.com/Qs-F/coloring`

## Usage

Import `github.com/Qs-F/coloring`, and use.

Example:  
```go
package main

import(
  "fmt"

  "github.com/Qs-F/coloring"
)

func main() {
  fmt.Printf("Hello, %s world!!", coloring.Red("coloring"))
}
```
